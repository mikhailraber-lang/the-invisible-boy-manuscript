# Narrative Rules

This directory contains repository-wide narrative and workflow rules for *The Invisible Boy*.

Rules canon exists to preserve:
- narrative integrity
- noir authenticity
- continuity discipline
- workflow consistency
- verification standards
- AI governance boundaries

---

# Rules Standards

Rules materials may include:

- noir tone guidance
- pacing expectations
- dialogue constraints
- POV restrictions
- violence boundaries
- continuity procedures
- verification protocols
- AI workflow governance
- repository discipline standards

Rules should remain concise, operational, and authoritative.

---

# Repository Rules

Narrative rules apply across:
- canon materials
- approved manuscripts
- verification workflows
- AI-assisted drafting systems

Major revisions should undergo:
- continuity review
- reconciliation review if necessary
- explicit author approval

---

# AI Interaction Rules

AI systems may:
- reference workflow rules
- validate procedural consistency
- detect continuity violations
- support verification processes

AI systems may NOT:
- override narrative governance
- reinterpret authorial intent automatically
- replace literary judgment
- bypass repository safeguards

---

# Canon Priority

When conflicts occur:

1. LOCKED repository rules
2. APPROVED governance materials
3. VERIFIED continuity systems
4. Draft workflow materials
5. Experimental AI outputs

---

# Literary Principle

Verification supports storytelling.

Verification does not replace storytelling.
