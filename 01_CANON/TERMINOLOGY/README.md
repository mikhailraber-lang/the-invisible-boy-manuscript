# Terminology Canon

This directory contains approved terminology and narrative naming conventions for *The Invisible Boy*.

Terminology canon exists to preserve:
- naming consistency
- narrative identity
- investigative clarity
- continuity stability
- terminology verification integrity

---

# Terminology Standards

Terminology materials may include:

- approved names
- locked terminology
- aliases
- prohibited substitutions
- investigative terminology
- institutional naming references
- recurring phrases
- continuity-sensitive wording

Format example:

Correct:
- "The Ghost"

Incorrect:
- "Grey Mouse"

Terminology references should remain concise and authoritative.

---

# Repository Rules

Approved terminology should remain stable across all verified materials.

Changes to locked terminology require:
- continuity review
- reconciliation review if necessary
- explicit author approval

Unauthorized synonym drift should be avoided.

---

# AI Interaction Rules

AI systems may:
- validate terminology consistency
- detect naming drift
- compare phrase usage
- identify continuity conflicts

AI systems may NOT:
- silently replace locked terminology
- introduce unauthorized aliases
- reinterpret approved naming conventions
- mutate recurring investigative references automatically

---

# Canon Priority

When conflicts occur:

1. LOCKED terminology canon
2. APPROVED canon
3. VERIFIED continuity materials
4. Draft manuscripts
5. Experimental AI outputs

---

# Literary Principle

Terminology shapes narrative identity.

Consistency strengthens immersion.
